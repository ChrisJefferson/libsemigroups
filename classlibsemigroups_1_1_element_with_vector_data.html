Detailed Description

template<typename S, class T>
class libsemigroups::ElementWithVectorData< S, T >

Abstract base class for elements using a vector to store their defining data.

The template parameter S is the type entries in the vector containing the defining data. The value of the template parameter S can be used to reduce (or increase) the amount of memory required by instances of this class.

The template parameter T is the subclass of ElementWithVectorData used by certain methods to construct new instances of subclasses of ElementWithVectorData.

For example, Transformation<u_int128_t> is a subclass of ElementWithVectorData<u_int128_t, Transformation<u_int128_t>>.
