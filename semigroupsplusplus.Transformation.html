ClassTransformation<T>

Template class for transformations, which is a subclass of semigroupsplusplus::PartialTransformation.

The value of the template parameter T can be used to reduce the amount of memory required by instances of this class.

A transformation f is just a function defined on the whole of {0, 1, ..., n - 1} for some integer n called the degree of f. A transformation is stored as a vector of the images of {0, 1, ..., n -1}, i.e. [(0)f, (1)f, ..., (n - 1)f].
