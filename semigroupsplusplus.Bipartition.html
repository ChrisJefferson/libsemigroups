Classsemigroupsplusplus::Bipartition

Class for bipartitions, which are partitions of the set {0, ..., 2n -1} for some integer n, which is a subclass of semigroupsplusplus::ElementWithVectorData.

The Bipartition, Bipartition and Bipartition class is more complex (i.e. has more methods) than strictly required by the algorithms for semigroupsplusplus::Semigroups because the extra methods are used in the GAP package Semigroups package for GAP.
