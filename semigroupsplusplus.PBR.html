Classsemigroupsplusplus::PBR

Partitioned binary relations are a generalisation of bipartitions, which were introduced by Martin and Mazorchuk. This is a subclass of semigroupsplusplus::ElementWithVectorData.

Bases

PBR ( std::vector<std::vector<unsigned int>, std::allocator<std::vector<unsigned int>>> * vector,
size_t hv )

Constructs a PBR defined by vector, vector is not copied, and should be deleted using semigroupsplusplus::ElementWithVectorData::really_delete and semigroupsplusplus::Element::really_delete.

The argument vector should be a vector of vectors of non-negative integer values of length 2n for some integer n, the vector in position i is the list of points adjacent to i in the PBR.

Pointer to defining data of the matrix.
