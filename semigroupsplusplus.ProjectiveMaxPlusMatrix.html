Classsemigroupsplusplus::ProjectiveMaxPlusMatrix

Class for projective square matrices semigroupsplusplus::MaxPlusSemiring semiring, which is a subclass of semigroupsplusplus::ElementWithVectorData. See semigroupsplusplus::Semiring. Two projective matrices are equal if they differ by a scalar multiple.

Constructs a matrix defined by matrix, matrix is not copied, and should be deleted using semigroupsplusplus::ElementWithVectorData::really_delete and semigroupsplusplus::Element::really_delete.

The argument matrix should be a vector of integer values of length n ^ 2 for some integer n, so that the value in position i n + j is the entry in the ith row and j*th column of the constructed matrix.
