Functions

Congruence *
parallel_todd_coxeter ( Congruence * cong1,
Congruence * cong2,
bool report )

This function can be used to run semigroupsplusplus::Congruence::todd_coxeter in parallel on two congruences. For example, if we want to compute a congruence on a semigroupsplusplus::Semigroup object, then we may want to run the Todd-Coxeter algorithm on an instance of semigroupsplusplus::Congruence where the coset table is prefilled with the left or right Cayley graph (where appropriate), and on another instance where the coset table is not prefilled.

When the semigroupsplusplus::Congruence::todd_coxeter method of one semigroupsplusplus::Congruence object terminates it kills the other method, semigroupsplusplus::Congruence::kill.

cong1

the first congruence to run semigroupsplusplus::Congruence::todd_coxeter
cong2

the second congruence to run semigroupsplusplus::Congruence::todd_coxeter
report

report during semigroupsplusplus::Congruence::todd_coxeter and this function itself, if any (defaults to semigroupsplusplus::DEFAULT_REPORT_VALUE)
return

either cong1 or cong2 depending on which of them the semigroupsplusplus::Congruence::todd_coxeter method finished first.
