ClassElementWithVectorData<S, T>

Abstract base class for elements using a vector to store their defining data. For example, semigroupsplusplus::Transformation<u_int128_t> is a subclass of ElementWithVectorData<u_int128_t, Transformation<u_int128_t>>.
