ClassPartialTransformation<S, T>

Template class for partial transformations, which is a subclass of semigroupsplusplus::ElementWithVectorData. For example, semigroupsplusplus::Transformation<u_int128_t> is a subclass of PartialTransformation<u_int128_t, Transformation<u_int128_t>>.

The value of the template parameter S can be used to reduce the amount of memory required by instances of this class.

This class is abstract since it does not implement all methods required by the semigroupsplusplus::Element class, it exists to provide common methods for its subclasses.

A partial transformation f is just a function defined on a subset of {0, 1, ..., n - 1} for some integer n called the degree of f. A partial transformation is stored as a vector of the images of {0, 1, ..., n -1}, i.e. [(0)f, (1)f, ..., (n - 1)f] where the value UNDEFINED is used to indicate that (i)f is, you guessed it, undefined (i.e. not among the points where f is defined).
