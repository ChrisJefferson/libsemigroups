Classsemigroupsplusplus::BooleanMat

Class for square matrices over the Boolean semiring, which is a subclass of semigroupsplusplus::ElementWithVectorData.

Bases

ElementWithVectorData<bool, semigroupsplusplus::BooleanMat>

Constructors

BooleanMat ( std::vector<bool> * matrix,
size_t hv )

Constructs a Boolean matrix defined by matrix, matrix is not copied, and should be deleted using semigroupsplusplus::ElementWithVectorData::really_delete and semigroupsplusplus::Element::really_delete.

The argument matrix should be a vector of boolean values of length n ^ 2 for some integer n, so that the value in position in + j is the entry in the ith row and jth column of the constructed matrix.

matrix

Pointer to defining data of the matrix.
hv

The hash value of the element being created.
