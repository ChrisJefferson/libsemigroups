114  private:
115  // Partial copy.
116  // \p copy a semigroup
117  // \p coll a collection of additional generators
118  //
119  // This is a constructor for a semigroup generated by the generators of the
120  // Semigroup copy and the (possibly) additional generators coll.
121  //
122  // The relevant parts of the data structure of copy are copied and
123  // \c this will be corrupt unless add_generators or closure is called
124  // subsequently. This is why this method is private.
125  //
126  // The same effect can be obtained by copying copy using the copy
127  // constructor and then calling add_generators or closure. However,
128  // this constructor avoids copying those parts of the data structure of
129  // copy that add_generators invalidates anyway. If copy has not been
130  // enumerated at all, then these two routes for adding more generators are
131  // equivalent.
132  Semigroup(Semigroup const& copy, std::vector<Element*> const* coll);
